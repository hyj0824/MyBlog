    еӯҗйӣҶжһҡдёҫ

    дёӨдёӘй—®йўҳпјҡ

    1. жңү \(n\) дёӘе…ғзҙ зҡ„йӣҶеҗҲпјҢжһҡдёҫе…¶еӯҗйӣҶ
    2. еҜ№дәҺиҜҘеӯҗйӣҶпјҢжһҡдёҫе…¶еӯҗйӣҶ

    Step 1 жһҡдёҫжңү \(n\) дёӘе…ғзҙ зҡ„йӣҶеҗҲ \(N\) зҡ„еӯҗйӣҶВ¶

    жҲ‘д»¬з”ЁдәҢиҝӣеҲ¶ж•°жқҘиЎЁзӨәжҹҗдёҖдёӘе…ғзҙ еҸ–жҲ–дёҚеҸ–пјҢд»ҘжӯӨжһ„жҲҗдёҖдёӘйӣҶеҗҲгҖӮ

    д»Һе…Ё1еҲ°е…Ё0пјҢдёҖе…ұ \(2^n\) дёӘгҖӮ

    for (int S = 0; S < (1 << n); S++)

    зҠ¶еҺӢdpз»Ҹеёёз”ЁпјҢиҝҷзҺ©ж„Ҹе„ҝжҳҜеҹәзЎҖж¬ҫ жһҡдёҫеӯҗйӣҶ

    жҲ‘д»¬з§° \(S\) иҝҷдёӘ \(n\) дҪҚдәҢиҝӣеҲ¶ж•°дёәдёҖдёӘзҠ¶жҖҒ

    Step 2 жһҡдёҫ \(S\)пјҲеҚі \(N\) зҡ„дёҖдёӘеӯҗйӣҶпјүзҡ„жүҖжңүеӯҗйӣҶВ¶

    з”ұдәҺдёҚж–№дҫҝзЎ®е®ҡ \(S\) дёӯз”ұ1йҖүдёӯзҡ„ \(k\) дёӘе…ғзҙ пјҢжқҘйҮҚж–°е»әз«ӢдёҖдёӘ\(2^k\)жһҡдёҫгҖӮеӣ жӯӨжҲ‘д»¬иҖғиҷ‘еҜ№иҝҷдёӘзҠ¶жҖҒ \(S\) еҒҡжүӢи„ҡгҖӮ

    дәҺжҳҜжңүпјҢжһҡдёҫеӯҗйӣҶпјҡ

    for (int T = S; T; T = (T - 1) & T) {
  // жҜҸиҪ®еҫӘзҺҜдёӯзҡ„TеҚідёәSзҡ„еӯҗйӣҶ
}

    зңҹеӯҗйӣҶпјҡ

    for (int T=(S-1)&S; T; T=(T-1)&S){
  // жҜҸиҪ®еҫӘзҺҜдёӯзҡ„TеҚідёәSзҡ„зңҹеӯҗйӣҶ
}

    жіЁж„Ҹ \(T\) зҡ„еӨ§е°ҸпјҢе…¶жһҡдёҫзҡ„йЎәеәҸдёҺ \(S\) зӣёеҸҚпјҢжҳҜ йҖҶеәҸ

    е…ідәҺдәҢиҝӣеҲ¶жһҡдёҫйӣҶеҗҲ - зҹҘд№Һ

    еҗҢж ·зҡ„пјҢе…¶ж—¶й—ҙеӨҚжқӮеәҰдёә \(O(2^{\text{popcount}(S)})\)пјҢдёӢиҜҒпјҡеҠ дёҠеӨ–йқўжҖ»еӨҚжқӮеәҰ \(O(3^n)\)гҖӮ

    Step 3 дёӨеұӮз»“еҗҲеҚіеҸҜ еӨҚжқӮеәҰиҜҒжҳҺВ¶

    е…ідәҺж—¶й—ҙеӨҚжқӮеәҰзҡ„иҜҒжҳҺпјҡhttps://www.cnblogs.com/CDOI-24374/p/15876755.html

    жҲ‘д»¬дёҚеҰЁиҖғиҷ‘жҢүйӣҶеҗҲзҡ„еӨ§е°Ҹ \(i\) жқҘжһҡдёҫз¬¬дёҖеұӮеӯҗйӣҶпјҡ\(\sum \limits_{i = 0}^n C^i_n = 2^n\)

    еҜ№дәҺеҗ«жңү\(i\)дёӘе…ғзҙ зҡ„еӯҗйӣҶпјҢжһҡдёҫжңүпјҡ\(2^i\)

    з»“еҗҲдёҖдёӢпјҢжңү\(\sum \limits_{i = 0}^n C^i_n 2^i\)пјҢд№ҹеҚіпјҡ

    \(\sum \limits_{i = 0}^n C^i_n 2^i*1^{n-i}\)

    дҪҝз”ЁдәҢйЎ№ејҸе®ҡзҗҶпјҡ

    \((a+b)^n = \sum \limits_{i = 0}^ n C^i_n a^i b^{n - i}\)

    жҜ”иҫғеҫ—пјҢ\(a=2,b=1\) зҡ„ж—¶еҖҷ \(3^n = \sum \limits_{i = 0}^n C^i_n 2^i\)гҖӮеӣ жӯӨжҳҜ \(O(3^n)\) зҡ„гҖӮ
